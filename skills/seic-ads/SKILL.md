# SEic 廣告決策 Skill

## 讀取品牌規範（執行前必做）
- `000_Agent/knowledge/STRATEGY.md` — 年度目標與指標
- `200_Reference/templates/SEic-brand-guide.md` — 品牌風格與受眾

---

## 第一步：確認模式

用戶開口時，先確認是哪個模式：

```
你好，這次要做哪個廣告檢視？

A. 週檢視（每週一，快速確認廣告健康狀態）
B. 月底檢討（月底，深度回顧 + 下月方向）
C. 月初提案（月初，產出完整廣告計畫）
```

## 模式 A｜週快速檢視

### 請提示用戶提供截圖

```
請提供以下 4 張截圖：

1. Ematic Meta 週報 → Overall Performance 那頁
2. Ematic Google 週報 → Campaign Overview 那頁
3. 業績週報 → 當週業績那頁
4. 行銷儀表板 → 最新版本

截圖丟過來即可。
如果本週有特殊狀況（新素材上線、活動開始/結束），
請另外告訴我一句話。
```

### 讀取截圖後，執行健康檢查

從截圖提取以下數據：

**Meta 讀取：**
- 總花費、ROAS、CTR、頻率、購買次數、CPP

**Google 讀取：**
- 總花費、ROAS、各活動花費與轉換、預約點擊數

**業績週報讀取：**
- 當週業績達成率、MoM%、廣告支出、ROAS（MER）、CAC

**行銷儀表板讀取：**
- 預約進店數、月達成率、進站總次數、IG/LINE 新增

### 健康檢查標準

| 指標 | 警戒線 | 動作 |
|------|-------|------|
| Meta ROAS | < 5x | 🔴 找原因，考慮暫停 |
| Meta CTR | < 1% | 🟡 素材開始疲乏，準備換 |
| Meta 頻率 | > 2.5 | 🟡 換素材或受眾 |
| Meta CPP | > $3,000 | 🔴 受眾或素材有問題 |
| Google PMAX | 預約點擊下滑 | 🟡 檢查廣告文案 |
| 任何廣告 ROAS | < 1x | 🔴 立刻暫停 |
| 業績 MoM% | 連續 2 週負成長 | 🔴 需要調整策略 |
| 預約達成率 | < 20% | 🔴 檢查預約頁面流程 |

## 模式 B｜月底深度檢討

### 請提示用戶提供截圖

```
請提供以下 5 張截圖：

1. Ematic Meta 月報 → Overall Performance Snapshot
2. Ematic Meta 月報 → Campaign Overview
3. Ematic Google 月報 → Campaign Overview
4. 業績週報 → 當月完整版
5. 行銷儀表板 → 當月完整版

截圖丟過來後，請補充一句話：
「下個月主題是___，庫存有壓力的車款是___」
```

### 月底檢討分析架構

**1. 廣告真實貢獻分析**
- 區分 MER（總業績/總廣告費）vs 各平台實際轉換
- 標注 Google SEM 品牌字的「攔截效應」（非新增需求）
- 找出真正帶來新客的廣告組

**2. 各廣告活動評分**

| 活動名稱 | 花費 | ROAS | 轉換 | 評級 | 下月建議 |
|---------|------|------|------|------|---------|
| ___ | ___ | ___ | ___ | 🟢/🟡/🔴 | 繼續/調整/砍掉 |

**3. 銷售 → 廣告對應邏輯**

```
熱銷車款（銷售台數排名前列）
├── 庫存充足 → 建議下月繼續投，可加碼
└── 庫存不足 → 暫停廣告，轉移預算到其他車款

滯銷車款（達成率偏低）
├── 新品 → 加強曝光，搭品牌內容
└── 舊款 → 提出消庫存方案，搭限時優惠

滯銷配件
└── 不單打，搭主力車款做組合包廣告
```

**4. 漏斗斷點診斷**
```
廣告點擊 → 進站 → 進預約頁 → 點擊預約按鈕 → 預約成功
找出哪個環節掉最多人 → 提出修復建議
```

**5. 必問反思 5 問**
```
1. 上個月廣告的唯一目標達成了嗎？
2. 有沒有廣告在打舊客卻用了新客預算？
3. 哪個廣告 ROAS < 1x 卻還在跑？
4. Paname / GoGreen 等活動廣告值得繼續付費打嗎？
   （還是改用 LINE 免費觸及就夠？）
5. 預約漏斗的瓶頸在廣告還是在頁面？
```

```
 庫存有壓力的是___，預算有沒有要調整___」
```

### 月初提案架構

**反思 5 問（產出提案前必問）**
```
1. 這個月廣告的唯一目標是什麼？（只能選一個）
   → 知名度 / 預約試乘 / 直接轉單

2. 主要打新客還是舊客？
   → 舊客優先用 LINE/Email，不用付費廣告

3. 有沒有大活動需要增加預算？
   → 如有，提出預算申請 + 預期 ROI

4. 哪台車款最值得主打？（根據上月銷售+庫存）

5. 上月哪個廣告要繼承？哪個要砍掉？
```

## 固定判斷規則（每次都適用）

### 預算動態調整觸發條件
```
↑ 增加預算：
  · ROAS > 50x 連續兩週
  · 大活動前兩週（廠拍、新品上市）
  · 預約進店達成率 > 60%

↓ 減少預算：
  · ROAS < 3x 且連續兩週
  · 某平台 CTR < 0.5% 且頻率 > 3

→ 跨平台轉移：
  · Meta 效果持續差 → 移給 Google SEM
  · Google PMAX 轉換歸零 → 移給 Meta 再行銷
```

### 不需要付費廣告的情境
```
以下情境改用免費管道（LINE / IG 有機貼文）：
· GoGreen 騎乘換紅利（打舊客，LINE 就夠）
· 活動回顧/結案（IG 貼文即可）
· 小型車主聚活動（LINE 群組推播）
· 純品牌形象（沒有明確 CTA）
```
