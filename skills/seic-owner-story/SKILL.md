# SEic 車主故事 Skill

## 讀取品牌規範（每次執行前必做）

執行前先讀這兩份文件：
- `000_Agent/knowledge/DESIGN.md` — 視覺風格與平台規格
- `200_Reference/templates/SEic-brand-guide.md` — 品牌個性、受眾、Tone of Voice

---

## 執行流程

### Step 1：收集資訊

如果用戶沒有主動提供，依序詢問以下資訊（一次問完，不要逐一問）：

```
1. 這個月的品牌主軸是什麼？（例如：母親節、夏日騎行、新品上市）
2. 這次的車主是哪個族群？（例如：新手爸媽、科技業上班族、樂齡族、毛孩家長）
3. 使用的車款是哪台？（Unimoke、miniu、UDX、CarryMe、Minibike 或其他）
4. 影片類型？
   - 短片（IG Reels / YT Shorts，60 秒內）
   - 長片（YouTube，3-10 分鐘）
5. 拍攝格式？
   - A. 獨白式（車主對鏡頭自己說話）
   - B. 訪談式（有人在旁邊問問題）
6. 這次的優先目標？
   - 知名度（讓更多人認識 SEic）
   - 好感（讓人覺得 SEic 很有品味）
   - 轉單（讓人想試乘或購車）
```

---

### Step 2：發想 3 個主題方向

根據收集到的資訊，產出 **3 個主題方向**，每個主題包含：

```
主題名稱：[簡短有力的標題，5-10 字]
故事核心：[這個故事想傳達的一句話]
主角設定：[什麼樣的車主，他的生活背景]
情感鉤子：[觀眾看完會有什麼感受]
適合時機：[什麼月份 / 節慶 / 情境最適合這個主題]
轉換方向：[這個故事如何自然帶出試乘 or 購車動機]
```

**主題篩選標準**（三個條件都要符合）：
1. 與品牌核心一致：「買車只是開始，我們想陪你走進一種生活」
2. 符合本月主軸或當下社會情境
3. 車主說出來是真實的，不是廣告話術

---

### Step 3：輸出腳本或訪綱

用戶確認主題後，根據拍攝格式輸出：

---

#### 格式 A：獨白式腳本（短片版，60 秒內）

```markdown
## 獨白式腳本 — [主題名稱]

**影片時長**：45-60 秒
**平台**：IG Reels / YT Shorts
**畫面建議**：[給剪輯師的場景提示]

### 給車主的說話方向（不是逐字稿，是引導）

開場（0-5 秒）：
→ [一句話切入，從生活場景開始，不要從「我叫XXX」開始]
→ 範例：「每個週末早上，我都比城市早醒來。」

故事展開（5-35 秒）：
→ 說明點 1：[車主生活中的某個問題或渴望]
→ 說明點 2：[電輔車如何出現在他的生活裡]
→ 說明點 3：[一個具體的生活場景改變]

收尾（35-50 秒）：
→ [一句有溫度的總結，不要說「SEic 真的很好」這種廣告話]
→ 範例：「它不只是一台車，是我每天切換身份的那個時刻。」

CTA（50-60 秒）：
→ 字幕：[預約試乘 / 連結在 Bio]（不要口說，視覺呈現即可）

### 注意事項
- 讓車主用自己的話說，不要背稿
- 如果車主卡住，提示：「說說你第一次騎 miniu 是去哪裡？」
- 情緒比資訊重要
```

---

#### 格式 A：獨白式腳本（長片版，3-5 分鐘）

```markdown
## 獨白式腳本 — [主題名稱]（長片）

**影片時長**：3-5 分鐘
**平台**：YouTube
**章節結構**：

Chapter 1（0:00-0:30）：開場 Hook
→ 最強的一句話或場景，讓人想看下去

Chapter 2（0:30-1:30）：車主背景
→ 他是誰？他的生活是什麼樣子？
→ 在買這台車之前，他有什麼「不滿足」或「渴望」？

Chapter 3（1:30-3:00）：轉折點
→ 為什麼選 SEic？（不要說規格，說感受）
→ 第一次騎的記憶

Chapter 4（3:00-4:30）：生活改變
→ 現在的生活裡，這台車扮演什麼角色？
→ 有沒有一個具體的故事或畫面？

Chapter 5（4:30-5:00）：結尾
→ 他想對還在猶豫的人說什麼？
→ CTA 字卡
```

---

#### 格式 B：訪談式訪綱

```markdown
## 訪談訪綱 — [主題名稱]

**影片時長**：[短片 60 秒 / 長片 5 分鐘]
**給訪問者的提示**：問完問題後讓車主說完整，不要打斷。

### 暖身問題（不錄影，讓車主放鬆）
1. 你通常週末會做什麼？
2. 你騎車多久了？

### 核心問題（這些才是鏡頭要拍的）

**開場**
Q1：可以用一句話形容你和這台車的關係嗎？

**故事**
Q2：在買這台車之前，你平常是怎麼移動的？
Q3：第一次騎 [車款] 是什麼感覺？有沒有哪個畫面讓你印象很深？

**生活改變**
Q4：現在它在你生活裡是什麼角色？（通勤？週末？載孩子？）
Q5：有沒有一個具體的故事，是你覺得「幸好我有這台車」的時刻？

**收尾**
Q6：如果你要對一個還在猶豫要不要試乘的人說一句話，你會說什麼？

### 備用追問句（車主說太簡短時用）
- 「可以多說一點嗎？當時的感覺是…？」
- 「那個畫面在哪裡？在做什麼？」
- 「你說的XX，你的家人/朋友有什麼反應？」

### 剪輯提示
- 選最有情緒的 3-5 段話
- 避免剪進規格介紹（讓觀眾去官網查）
- 結尾留 Q6 的回答
```
