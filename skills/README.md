# SEic AI Skills

Claude Code Skills 供 SEic / Urban Drivestyle 行銷團隊使用。

## 安裝方式

將 `skills/` 資料夾內的子資料夾複製到你的 `~/.claude/skills/`：

```bash
cp -r skills/* ~/.claude/skills/
```

## 使用方式

在 Claude Code 對話中說出觸發口令即可啟動對應的 Skill。

---

## 可用 Skills

### `seic-ads` — 廣告決策與提案
**觸發**：「廣告檢視」、「週檢視」、「月底檢討」、「廣告提案」、「廣告數據分析」

分三個模式：週快速檢視 / 月底深度檢討 / 月初廣告提案。

---

### `seic-blog` — 官網部落格 SEO 文章
**觸發**：「幫我寫部落格」、「寫一篇 SEO 文章」、「官網文章」、「幫我想部落格主題」

輸出格式：SEO On 填表資訊（直接複製貼上）＋ 文章本體。目標每篇 80 分以上。

---

### `seic-owner-story` — 車主故事腳本
**觸發**：「幫我想車主故事主題」、「幫我寫訪綱」、「下個月要拍車主故事」

輸出格式：獨白式（車主自說）或訪談式（有人問問題）。

---

### `seic-reply` — 社群留言回覆
**觸發**：「回覆留言」、「幫我回這則留言」

風格：接住用戶說的話，讓笑點自己落地。不煽情、不說教。
